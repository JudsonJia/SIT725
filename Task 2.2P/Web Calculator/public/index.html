API Endpoints

You can test these endpoints directly in your browser or using tools like Postman:

GET Requests (URL Parameters):

GET /add/5/3 → Add 5 + 3
GET /subtract/10/4 → Subtract 10 - 4
GET /multiply/6/7 → Multiply 6 × 7
GET /divide/15/3 → Divide 15 ÷ 3

GET Requests (Query Parameters):

GET /add?num1=5&num2=3 → Add using query params

API Documentation:

GET /api → View all available endpoints
